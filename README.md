# PEdit

PEdit is a lightweight, tabbed text editor built with C# and WPF. It is designed to be straightforward and responsive, offering standard text editing capabilities along with practical features for viewing and managing code.

## Main Features

* **Tabbed Interface:** Open, edit, and manage multiple files in a single window.
* **File Explorer:** A built-in folder tree view for navigating directories and opening files quickly.
* **Search & Replace:** Supports plain text and Regular Expression searches, with options to find or replace within the current file or across all open tabs.
* **Live Markdown Preview:** Includes a separate, real-time preview window for rendering Markdown files into HTML.
* **Drag & Drop:** Load multiple files or directories simply by dragging them into the editor.
* **Syntax Highlighting:** Optional code coloring for known file extensions (can be toggled in the options).
* **Customizable Workspace:** Adjust font family, font size, text color, and background color.
* **Dark UI:** A native dark theme applied to the entire interface, including context menus and scrollbars.
